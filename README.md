# Python Selenium QA Automation

## 🇬🇧 English

### Description

This project is a simple **web login application built with Flask** and automated tests using **Pytest and Selenium**.
It was created to practice **QA Automation and UI testing with Python**.

### Technologies

* Python
* Flask
* Pytest
* Selenium
* Edge WebDriver

### Project Structure

```
src/
    app.py
templates/
    login.html
tests/
    test_login.py
    test_login_ui.py
requirements.txt
```

### How to Run the Project

Install dependencies:

```
pip install -r requirements.txt
```

Run the web application:

```
python src/app.py
```

Run automated tests:

```
pytest
```

### Test Types

* Unit Tests
* Functional Tests
* UI Automation Tests
